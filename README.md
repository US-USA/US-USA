# TikTok Story Viewer in Python

📢 Content & updates via Telegram: [@z_0_g](https://t.me/z_0_g)

## Features

- View TikTok stories without logging in
- Fetch story details including:
  - Video URL
  - Cover image
  - Heart count
  - View count
  - Timestamp
